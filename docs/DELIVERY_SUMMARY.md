## 📝 使用说明

### 方式一：Web 界面
1. 访问 `http://localhost:5000`
2. 上传 CSV/Excel 文件
3. 配置质量规则
4. 点击"开始测评"
5. 查看 HTML 报告

### 方式二：API 调用
```python
import requests

# 上传文件
response = requests.post('http://localhost:5000/api/upload',
                       files={'file': open('data.csv', 'rb')})
file_id = response.json()['file_id']

# 执行评估
rules = [{"column": "id", "rule_type": "unique"}]
response = requests.post('http://localhost:5000/api/evaluate',
                       json={"file_id": file_id, "rules": rules})
print(response.json()['report_url'])
```

### 方式三：演示脚本
```bash
python demo.py
```
