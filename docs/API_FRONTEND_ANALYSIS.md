## 二、前端功能缺失清单

### 2.1 高优先级缺失功能（P0）

1. **资产编辑功能** - assets.js中调用了PUT API，但界面没有"编辑"按钮触发
2. **规则编辑功能** - asset_detail.js中editRule()函数只是显示警告
3. **执行校验功能** - 没有界面触发校验执行
4. **调度配置功能** - 完全未实现界面

### 2.2 中优先级缺失功能（P1）

1. **批量问题操作** - 前端按钮已添加，但API调用被注释
2. **异常数据下载** - API已定义，前端未调用
3. **告警配置管理** - 完全未实现

### 2.3 低优先级缺失功能（P2）

1. **质量报告生成** - 旧版API `/api/evaluate` 需要迁移到新版
2. **资产详情页增强** - 编辑、快速操作等功能
